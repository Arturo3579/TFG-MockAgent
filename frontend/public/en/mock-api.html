What is a Mock API?

A mock API (or fake API) is a simulated programming interface that mimics the behavior of a real API but without a backend. It returns predefined responses with the status codes, headers and JSON that you configure.

It is the perfect tool for developers who need to test applications, prototype interfaces or develop AI agents without depending on expensive or unstable external services.

Mock API vs Dummy API

There are several related terms:

  • Mock API: Simulates the complete behavior of a real API with routes, methods and responses.
  • Dummy API: Generally returns simple, static data without logic. More basic.
  • Fake API: Broad term that includes mocks and dummies. Used interchangeably.
  • Stub API: Returns predefined responses for unit tests. Simpler than a mock.

¿Qué es un Mock Server?

Un mock server (o servidor mock) es un servidor simulado que imita el comportamiento de una API real. En lugar de conectarte a un backend costoso o inestable, el mock server devuelve respuestas predefinidas con el status code, headers y body JSON que tú configures.

Es la herramienta esencial para desarrolladores que trabajan con APIs REST, microservicios o agentes de IA que necesitan datos de prueba sin depender de servicios externos.

¿Para qué sirve un Servidor Mock?

🧪 Testing sin backend

Testea tu frontend o agente de IA sin necesidad de que el backend esté listo. Simula errores 500, 404, 401 y casos edge.

⚡ Prototipado rápido

Valida ideas en minutos. Crea un endpoint mock con JSON de respuesta y comparte la URL con tu equipo.

💰 Reduce costes

Evita llamadas costosas a APIs de terceros (Stripe, OpenAI, etc.) durante desarrollo. Ahorra hasta un 65% en costes.

🤖 Agentes de IA

Desarrolla agentes autónomos que consumen APIs sin depender de servicios reales. Itera más rápido.

Cómo crear un Mock Server en 3 pasos

1. Define tu endpoint

Elige la ruta de tu API (ej: /api/v1/users), el método HTTP (GET, POST, PUT, DELETE) y el código de status que quieres simular.

2. Configura la respuesta JSON

Pega tu JSON de respuesta en el editor. Valida la sintaxis en tiempo real. Puedes simular desde una simple lista de usuarios hasta respuestas complejas con arrays anidados.

3. Obtén tu URL

Publica tu mock y recibe una URL pública instantánea como https://tfg-mockagent-production.up.railway.app/mock/tu-ruta. Úsala en tu agente de IA, frontend o tests.

Mock Server vs API Real

¿Por qué usar un mock server en vez de una API real durante desarrollo?

  • Coste: Las APIs reales (Stripe, OpenAI, etc.) cobran por llamada. Un mock server es gratuito.
  • Velocidad: Respuesta en <10ms vs segundos de una API real.
  • Estabilidad: Tu mock server nunca cae. Las APIs reales sí.
  • Control: Genera errores a demanda para testear casos edge.
  • Privacidad: No envías datos reales a terceros durante desarrollo.

¿Es realmente gratuito?

¿Qué diferencia hay entre mock server, stub y fake API?

Un mock server simula todo el comportamiento del servidor (rutas, status, headers). Un stub es más simple (solo devuelve datos). Fake API es un término más general. MockAgent.AI es un mock server completo.

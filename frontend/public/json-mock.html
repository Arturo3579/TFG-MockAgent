¿Qué es un JSON Mock?

Un JSON mock es una respuesta en formato JSON simulada que imita los datos que devolvería una API real. Es el componente fundamental de cualquier mock server o API de prueba.

En vez de esperar a que el backend esté listo, generas un JSON con la estructura exacta que necesitas y tu frontend o agente de IA lo consume como si fuera real.

Ejemplo de JSON Mock

Imagina que necesitas simular una API de usuarios. Con MockAgent.AI creas este JSON:

{ "users": [ { "id": 1, "name": "Ana García", "email": "ana@example.com", "role": "admin", "active": true }, { "id": 2, "name": "Carlos López", "email": "carlos@example.com", "role": "user", "active": false } ], "total": 2, "page": 1 }

Y obtienes una URL pública: https://tfg-mockagent-production.up.railway.app/mock/api/users

¿Para qué sirve un Generador JSON Mock?

🎨 Frontend Development

Desarrolla tu UI con datos reales desde el día 1. No esperes al backend para ver cómo se renderiza tu lista, tabla o gráfico.

🧪 Testing

Testea casos edge: listas vacías, arrays enormes, objetos anidados, valores null. Todo sin depender de una base de datos real.

🤖 Agentes de IA

Entrena tu agente con datos JSON estructurados. Simula APIs de CRM, ERP, e-commerce o cualquier servicio que tu agente necesite consumir.

📱 Prototipado

Valida ideas con stakeholders mostrando datos reales. Un JSON mock hace que tu prototipo se vea funcional y profesional.

Tipos de JSON Mocks que puedes crear

JSON Mock Simple

Un objeto básico con pocos campos. Ideal para tests rápidos y endpoints simples.

JSON Mock con Arrays

Listas de objetos JSON. Perfecto para simular colecciones de usuarios, productos, pedidos, etc.

JSON Mock Anidado

Objetos dentro de objetos. Útil para APIs complejas con relaciones (usuario → empresa → departamentos).

JSON Mock con Errores

Simula errores de validación con status 400 y JSON de errores: {"errors": [{"field": "email", "message": "Invalid format"}]}

¿Cómo crear un JSON Mock con MockAgent?

  • Entra a MockAgent.AI (sin registro necesario para Mock Express)

Preguntas frecuentes sobre JSON Mocks

¿El JSON mock tiene que ser válido?

Sí, el editor de MockAgent valida la sintaxis JSON en tiempo real. Si hay errores, te lo indica antes de publicar.

¿Puedo cambiar el JSON después de publicar?

Sí, con cuenta gratuita. Edita el endpoint, modifica el JSON y guarda. La URL se mantiene igual.

¿Puedo generar JSON dinámico?

Actualmente es estático. En el futuro añadiremos templates dinámicos con variables y generación aleatoria.

¿Es gratuito el generador JSON mock?
